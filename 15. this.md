## this

- is created when running code.
- depends on where it is or how the functions are run.

1.
```
function a(){

   console.log(this);

}

a();   
```
=> this = window

2.
```
var b= function (){

  console.log(this);

}

b();
```
=> this = window

3.
```
var c={

   name: the c object;

   log: function(){

   console.log(this);

}

}

c.log();
```
=>this = object c

- so we can say:   
this, when inside a function, is referring to the environment where the function sits.
