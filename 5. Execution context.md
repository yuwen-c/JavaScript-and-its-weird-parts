## Hoisting

- put this code in the previous file:

```
var a = 'Hello World!';

function b() {
    console.log('called b!')
}

b();
console.log(a);
```
  - it runs perfect.

- Then change it's order:

```
b();
console.log(a);

var a = 'Hello World!';

function b() {
    console.log('called b!')
}

```
  - Weird thing happens,
  function b() runs, and variable a is undefined.
  there is no error.
  
- if we try this:
```
b();
console.log(a);

// var a = 'Hello World!';

function b() {
    console.log('called b!')
}

```
  - we'll see a truly error message.
