## namespace

- Javascript does not have namespace concept.
- however, we can fake it with object.

```
var greet= "hello";
var greet= "hola";
```
collision.

```
var english={};
var spanish={};

english.greet="hello";
spanish.greet="hola";
```

But, we can not just say:

```
english.greeting.greet="hello";

```
because the left part is undefined.
so we must say

```
english.greeting={};
english.greeting.greet="hello";

```
or:

```
var english={

  greeting:{
    greet:"hello"
  }
};

```
