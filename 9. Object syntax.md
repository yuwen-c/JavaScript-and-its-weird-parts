## object

object is name, value pair, seperated by colon":".

```
var Tony = { 
firstname:"Tony",
lastnsme:"Alicea"
}
```

```
function greet(person)={

console.log("hi,"+ person.firstname)
}

greet(Tony);

```



```
greet({
firstname:"Sara",
lastnsme:"Pop"
});


```

I can also create an object on the fly.
